# Examples
{: .no_toc }

This section provides complete, working examples that demonstrate various use cases for YAMLtecture. Each example includes the full YAML configuration, any queries or Mermaid settings used, and the resulting output. All examples are validated as part of the project's test suite.

---

| Example | Description |
|:--------|:------------|
| [Microservices Architecture](/examples/microservices) | A basic microservices setup with an API gateway, services, and databases connected via REST, gRPC, and DB links. |
| [Cloud Infrastructure](/examples/cloud-infrastructure) | A hierarchical cloud architecture using parent-child relationships and Mermaid subgraphs to visualize network topology. |
| [Data Pipeline](/examples/data-pipeline) | A data processing pipeline from ingestion to dashboard, demonstrating query filtering to extract a subset of nodes. |
| [Multi-File Configuration](/examples/multi-file) | Splitting a configuration across multiple YAML files and merging them into a single architecture definition. |
| [Styled Mermaid Diagram](/examples/styled-diagram) | Customizing Mermaid output with node styles, link styles, and subgraphs for a polished diagram. |
